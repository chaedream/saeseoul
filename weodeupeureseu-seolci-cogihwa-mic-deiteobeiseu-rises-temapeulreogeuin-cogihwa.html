워드프레스 설치 초기화 및 데이터베이스 리셋, 테마/플러그인 초기화 

워드프레스 설치 초기화 및 데이터베이스 리셋, 테마/플러그인 초기화

1. 워드프레스 데이터베이스 내용 삭제하기

먼저 플러그인 추가한다(Plugin - Add Plugins) WP Datsbase Reset 플러그인을 설치(Install)하고 활성화(Active) 한다.

설치된 플러그인에서 Reset Database 링크를 선택 팝업창 뜨면 가장 하단 링크 선택(I don't ...) 2번 영역에서 Select All Tables 선택하고 아래 체크박스 해제하고 3번 영역에서 화면에 나온 보안코드 입력하면 나타나는 Reset Tables 버튼 클릭 확인 메시지 창에서 'OK' 선택

리셋 작업 끝나면 어드민 계정으로 로그인 가능(어드민 계정은 삭제되지 않음) 현재 활성화된 테마를 확인하고 꼭 기억해둘 것!!! (아직 테마 파일들은 지워지지 않은 상태) 나중에 테마 파일을 삭제할 때 활성화된 테마 파일은 절대 삭제하면 안된다.

2. FTP 접속하야 테마와 플러그인 및 업로드 데이터 삭제하기

이제 FTP로 워드프레스 서버에 접속해서 불필요한 파일들을 정리해야 한다. 파일질라 같은 FTP 프로그램을 설치한다. 까페24 등에 워드프레스 서버를 신청했다면 ftp 접속 비밀번호를 설정했을 것이다. FTP 접속에 성공하면 서버 폴더 목록이 나타난다.

wp-contents 안의 plugins 폴더 선택하고 index.php를 제외한 나머지 파일/폴더를 삭제한다. themes 폴더에서 기본 테마와 활성화된 테마, index.php 파일을 제외한 나머지 테마 폴더를 삭제한다. upload 폴더에 있는 모든 파일과 폴더를 삭제한다. wp-contents에서 plugins, themes, upload 폴더, index.php 파일을 제외한 나머지 폴더를 삭제한다. (필요한 폴더가 있으면 남겨도 된다)

3. 확인 워드프레스 관리자페이지에서 초기화 작업이 잘 되었는지 확인한다.
