웹 사이트에 엑셀 데이터 또는 스프레드 시트 데이터를 표시하는 방법

비교 테이블을 공유하거나 제품 정보를 공유하려는 경우 테이블 또는 WordPress Excel 스프레드 시트 플러그인을 사용하면 더 효과적으로 작업을 수행 할 수 있습니다.

스포츠 웹 사이트 또는 영화, 웹 사이트를 소유하고있는 경우 WordPress Excel 스프레드 시트 플러그인을 사용하여 라이브 데이터를 표시 할 수 있습니다. 이러한 라이브 데이터는 사이트를 새로 고치거나 다시로드하지 않고도 업데이트됩니다.

스프레드 시트 플러그인은 이벤트 웹 사이트에서 남은 티켓 수를 표시하거나 좌석 계획 및 이벤트에 참여한 사람 수를 공유하는데도 유용합니다. 전반적으로 WordPress Excel 스프레드 시트 플러그인을 사용한다고 말할 수 있습니다. 방문자에게 데이터를 제공 할 수 있습니다.

게다가 페이지에 스프레드 시트를 추가하고 스프레드 시트를 편집하는 것은 플러그인을 사용하지 않고 어렵습니다. 그리고 WordPress는 플러그인을 사용하지 않고 스프레드 시트 내에서 라이브 데이터를 표시 할 수 없습니다.

오늘 저는 게시물에 데이터 또는 스프레드 시트를 표시하고 스프레드 시트를 편집 할 수있는 최고의 WordPress Excel 스프레드 시트 플러그인 7 개를 직접 선택했습니다

7 Best Free WordPress Excel Spreadsheet Plugins

1. TablePress

TablePress는 Excel, CSV, HTML 및 JSON 파일에서 테이블을 가져오고 내보낼 수있는 가장 인기있는 WordPress Excel 스프레드 시트 플러그인 중 하나입니다. 백엔드에 스프레드 시트와 같은 인터페이스가 있어 단일 코드 라인을 작성하지 않고도 데이터를 동시에 추가 할 수 있습니다.

TablePress를 사용하면 고유 한 표를 만들고 단축 코드를 통해 게시물, 페이지 및 위젯 영역에 이러한 표를 삽입 할 수 있으며 필터, 페이지 매기기 및 정렬을 지원합니다. 이미지와 링크를 삽입 할 수도 있습니다.

이 플러그인은 콘텐츠를 더 잘 제어 할 수있는 고급 편집기를 제공합니다.

이 WordPress 엑셀 스프레드 시트 플러그인은 사용자 친화적입니다.

거의 모든 사용자가 플러그인을 사용할 수 있습니다.

테이블의 데이터를 쉽게 조정할 수 있습니다. 특정 행과 열을 선택한 후 숨기고, 표시하고, 복제하고, 새 셀을 삽입하고 삭제할 수 있습니다.

TablePress를 사용하여 만든 테이블은 다음과 같습니다. 여기 에서 더 많은 데모를 확인 하세요.

이 플러그인은 기능을 다른 수준으로 확장하는 확장 기능도 지원합니다. 대부분의 확장은 무료입니다.

주요 기능

  • 무료 핵심 플러그인
  • 거의 모든 형식 지원
  • 단축 코드 지원 Shortcode
  • 스타일 변경을 위한 고급 편집기
  • 테이블 미리보기
  • 사이트 대시 보드에서 표 관리 및 수정
  • CSS로 테이블을 개별적으로 사용자 지정

가격 : 플러그인의 핵심 버전은 무료 입니다. 그러나 프리미엄 확장을 확인할만큼 궁금하다면 구매 해야합니다. 개발자는 일부 연장에 기부 금액을 권장하고 기타 프리미엄 연장에 대해서는 금액을 권장하지 않습니다. 약간의 기부가 필요합니다.

  

2. Inline Google Spreadsheet Viewer

Google 스프레드 시트, CSV 파일 및 MySQL 데이터베이스에 저장된 데이터를 대화 형 차트 또는 인포 그래픽으로 변환 하시겠습니까? 이 경우 인라인 Google 스프레드 시트 뷰어가 검색중인 플러그인입니다.

Google 스프레드 시트, Google Apps 스크립트, WordPress 데이터베이스, 원격 MySQL 등과 같은 다양한 소스의 데이터를 자유롭게 사용할 수 있습니다. CSV 파일도 업로드 할 수 있습니다.

이 WordPress 엑셀 스프레드 시트 플러그인은 Google 스프레드 시트, CSV 파일 등을 사이트에 직접 삽입하는 데 가장 적합합니다. 제공된 단축 코드를 사용하여 소스에 연결하기 만하면됩니다. 원본 파일이 변경되면 플러그인이 웹 사이트에서 변경 사항을 자동으로 업데이트합니다.

이 플러그인은 사용자 정의 관점에서 중요하지 않지만 여러 타사가 정기적으로 업데이트하는 테이블과 스프레드 시트를 사이트에 포함하는 경우 확인하는 것이 좋습니다.

특장점

  • 기본 소스가 변경되면 테이블 데이터 자동 업데이트
  • 데이터를 환상적인 차트 또는 인포 그래픽으로 변환
  • 포함을 위해 거의 모든 온라인 문서 파일 지원
  • 다양한 소스의 데이터 사용
  • 표 및 차트 모양 표
  • 다양한 차트 유형

가격 : 이 플러그인을 무료로 즐기십시오 .

3. Import Spreadsheets from Microsoft Excel

이름에서 알 수 있듯이 Microsoft Excel에서 스프레드 시트 가져 오기를 사용하면 번거 로움없이 MS-Excel에서 실시간 계산 스프레드 시트를 추가하거나 가져올 수 있습니다. 이 플러그인은 HTML 및 JavaScript로 라이브 온라인 스프레드 시트를 만드는 Excel 컴파일러와 비슷합니다.

이 WordPress 엑셀 스프레드 시트 플러그인을 사용하여 가격 계산, 재무 계산, 투자 수익률 계산, 건강 계산기 등과 같은 아름다운 웹 앱을 만들 수도 있습니다. 또한 자동 데이터 유효성 검사를 사용하여 스마트 웹 양식을 만들 수도 있습니다. 이 플러그인은 수식 및 함수를 포함하여 웹에서 Excel의 모든 기능을 자유롭게 사용할 수 있도록합니다.

기억해야 할 한 가지는 파일을 웹 형식으로 변환 할 수 있도록 Microsoft Excel 용 스프레드 시트 변환기 추가 기능을 사용해야한다는 것입니다. 이 추가 기능은 MS Excel 2007에서 이후 버전까지 작동합니다.

특장점

  • 계산 스프레드 시트 가져 오기
  • 그것으로 웹 앱을 만들 수 있습니다
  • 자동 유효성 검사 기능이있는 스마트 양식
  • 모든 유형의 스프레드 시트 형식 지원
  • 플러그인으로 만든 앱은 모바일 친화적입니다.
  • 양식 제출을위한 파일 첨부 허용
  • 가입 목록 및 기타 공유 양식 생성

가격 : 이 WordPress 엑셀 스프레드 시트 플러그인을 무료로 다운로드하십시오 .

4. Report Builder

Report Builder by wpDataTables는 한 번의 클릭으로 워드 프레스 사이트에서 워드 또는 엑셀 문서를 생성 할 수있는 wpDataTables의 프리미엄 애드온입니다. 이 플러그인은 wpDataTables를 데이터 공급자로 사용하고 스냅 내에서 문서를 생성합니다.

이 애드온은 계약, 송장, 다양한 통계 및 보고서 생성에 가장 적합합니다. 이 애드온을 사용하여 거래 목록, 환영 편지, 상업 제안, 시험 목록 등을 생성 할 수도 있습니다.

이 애드온을 사용하여 생성 된 보고서는 CSS 클래스를 사용하여 추가로 사용자 정의 할 수 있으며 표준 WordPress 비주얼 편집기 또는 Visual Composer를 사용하여 페이지에 쉽게 삽입 할 수 있습니다.

또한이 애드온은 사용자 친화적이므로 구성이나 설정이 전혀 없습니다. 단 한 번의 클릭으로 100 개의 파일을 생성하고 다운로드 할 수 있습니다. 보고서가 생성 될 DOCX 또는 XLSX 템플릿을 제공하기 만하면됩니다.

특장점

  • MS 워드 DOCX
  • 무제한 프런트 엔드 입력
  • 합계 계산
  • 대량 보고서 생성
  • 페이지에 쉽게 삽입
  • MS Excel XLSX 보고서 생성
  • 완전히 문서화 됨
  • 더 나은 지원 및 업데이트
  • 사용자 친화적
  • 제로 구성

가격 : 이 애드온을 사용하려면 wpDataTables를 사용해야하고이 프리미엄 애드온을 Codecanyon 에서 $ 175에 구입해야합니다 .

5. Ipushpull

Ipushpull은 단 몇 번의 클릭만으로 웹 사이트에 엑셀 데이터를 표시하는 강력한 플러그인입니다. 이 플러그인은 사이트에 라이브 업데이트 데이터를 표시하는데도 가장 적합합니다. 예를 들어 모든 스포츠의 자세한 실시간 점수를 표시 할 수 있습니다. 제 3 자 또는 귀하가 소스 데이터를 수정하면 데이터가 자동으로 변경됩니다.

먼저 MS Excel에 Ipushpull 추가 기능을 설치 한 다음 데이터를 선택하고 Ipushpull 서버에 저장해야합니다. 사이트에 데이터를 표시하려면 플러그인을 설치하고 단축 코드를 통해 페이지 및 게시물의 데이터를 가져와야합니다. 사이트에 엑셀 데이터를 표시하는 것은 간단합니다.

페이지를 다시로드하지 않고 데이터가 자동으로 업데이트되므로 사이트에 표시하는 라이브 콘텐츠는 사용자 페이지를 새로 고칠 필요가 없습니다. 표시 할 셀 범위를 선택할 수 있습니다. 또한 데이터 업데이트 빈도를 설정하고 데이터를 공개하거나 암호로 보호 할 수 있습니다.

특장점

  • Excel에서 데이터 포함
  • 라이브 데이터 업데이트
  • WP 단축 코드 생성기
  • Excel의 차트
  • 100 % 반응
  • 비밀번호 보안
  • 주파수 제어 업데이트
  • 페이지를 다시로드하지 않고 데이터를 업데이트합니다.

가격 : 플러그인은 무료 이지만 플러그인 을 사용하려면 Ipushpull 계정이 필요합니다. 이 WordPress 엑셀 스프레드 시트 플러그인을 사용하면 계정을 생성 한 후 무료 평가판을 시작할 수 있습니다. 그러나 무료 평가판 이후에는 10 명의 사용자를 위해 월 $ 250에 플러그인 을 구입 해야합니다. 플러그인을 구매하기 전에 가격 비교를 확인하십시오.

6. League Table

League Table은 다중 사이트 및 다국어 지원과 함께 정렬 가능한 열을 제공하는 최고의 프리미엄 테이블 솔루션 중 하나입니다. 이름에 따라 LeagueTable 플러그인은 스포츠 사이트를 위해 만들어졌습니다. 그러나 고도로 사용자 정의 가능한 옵션으로 인해 모든 종류의 웹 사이트에서 플러그인을 사용할 수 있습니다.

직관적 인 인터페이스를 통해 모든 유형의 사용자가 플러그인을 사용하여 테이블을 쉽게 만들 수 있습니다. 데이터를 복사하여 플러그인 편집기에 붙여 넣으면 30 초 이내에 테이블을 만들고 게시 할 수 있습니다. CSV 파일을 가져 오는 번거 로움이 없습니다.

이 플러그인은 사용자 정의 색상, 글꼴, 테두리, 반응 형 레이아웃 등과 같은 135 개 이상의 사용자 정의 옵션으로 가득 차 있습니다. 셀의 크기를 편집 할 수도 있습니다. 원하는 테이블을 만든 후 페이지에 추가하거나 단축 코드를 통해 게시하십시오.

특장점

  • 다국어 지원
  • 다중 사이트 지원
  • 단축 지원
  • 100 % 반응
  • 직관적 인 스프레드 시트 편집기
  • 정렬 가능한 열
  • 셀 속성
  • 수입 수출
  • 더 나은 지원

가격 :  이 모든 최고 수준의 기능은 $ 39에 대해서만 제공됩니다. 이 플러그인의 비용은 목록에서 가장 저렴한 프리미엄 플러그인입니다. Codecanyon 에서 저렴한 가격으로 WordPress Excel 스프레드 시트 플러그인을 구입하십시오 .

7. WP Sheet Editor

마지막으로 WP Sheet Editor는 WordPress에서 스프레드 시트를 열고 모든 정보를 표시하며 정보를 대량으로 편집 할 수있는 또 다른 최고의 WordPress 스프레드 시트 플러그인입니다. 이 플러그인을 사용하여 제품이나 정보를 하나씩 편집하고 중요한 문제를 처리하는 데 더 많은 시간을 투자하십시오.

이 플러그인을 사용하면 Excel 파일 또는 Google 시트를 내보내고 가져올 수 있으며 실시간으로 편집 할 수 있습니다. 이 플러그인은 WooCommerce 웹 사이트를 소유하고있을 때 유용합니다. 모든 제품의 사소한 정보를 편집해야 할 때 하나씩 편집하는 것은 시간이 많이 걸리고 번거 롭습니다. 스프레드 시트에서 여러 제품의 이미지를 동시에 추가 할 수도 있습니다.

WP Sheet Editor를 사용하면 스프레드 시트 내에서 제품 목록을 볼 수 있으며 여기서 제품의 모든 사소한 문제를 변경하거나 정보를 대량으로 추가 할 수 있습니다. 구텐베르크 사용자라면 좋은 소식이 있습니다. 이 플러그인을 사용하면 스프레드 시트에서 바로 Gutenberg 편집기를 사용하여 게시물 내용을 편집 할 수 있습니다.
