Gabia를 이용해 EC2 public ip를 호스팅하는 방법

AWS에서도 도메인을 관리하고 호스팅해 주는 서비스를 제공하지만, 개발용으로 저렴이 이용하기 위해 Gabia를 이용해 EC2 public ip를 호스팅하는 방법을 정리해보았습니다.

​

아래 링크로 이동하면 쉽게 사용가능한 도메인을 확인해서 신청할 수 있습니다. 당연히 잘 알려지지 않은 주소로 신청해야 아래와 같이 저렴한 가격에 1년을 사용할 수 있습니다.

​

https://www.gabia.com/

가비아: 국내 1위

도메인, 호스팅, 서버, 클라우드, 기업솔루션, 홈페이지, 쇼핑몰

www.gabia.com

​

원하는 도메인을 선택하고 신청 -> 결제 까지 완료하면 My가비아에서 관리툴 버튼을 클립합니다.

설정 메뉴에서 DNS 레코드 설정 메뉴의 설정 버튼을 클릭합니다.

A레코드 타입으로 www, @ 호스트로 EC2 public ip 를 등록합니다.

​

정상적으로 등록이 되었는지는 nsloolup 명령을 통해 확인할 수 있습니다. nslookup 명령으로 확인한 도메인의 주소가 EC2의 public ip 주소가 맞는지 확인합니다.
