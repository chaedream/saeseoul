.NET machine.config 오류 - (0xC00CE508 파서 리턴 오류)

Net & Program/C#.Net

[오류메시지]

C:\Windows\Microsoft.NET\Framework64\v2.0.50727\config\machine.config Parser returned error 0xC00CE508

OR

C:\Windows\Microsoft.NET\Framework\v2.0.50727\config\machine.config Parser returned error 0xC00CE508

[해결방법]

1. 메세지에 나타난 machine.config 폴더에 간다.

> C:\WINDOWS\Microsoft.NET\Framework\v2.0.50727\CONFIG 또는

> C:\WINDOWS\Microsoft.NET\Framework64\v2.0.50727\CONFIG

2. machine.config 파일을 찾는다.

3. 해당파일을 machine.config.old 로 파일명을 변경한다.

4. 같은 폴더내에 machine.config.default 파일을 machine.config 로 파일명을 변경하여 복사한다.

5. 실행시 오류난 프로그램을 재설치한다.

--------------------------------------------------------------------------

[원문]

In the folder C:\WINDOWS\Microsoft.NET\Framework\v2.0.50727\CONFIG 1.Find the file "machine.config". 2.Rename the file to "machine.config.old". 3.Copy "machine.config.default" to "machine.config" in the same folder. 4.Install the Windows Companion.

Also verify that the latest version of the .NET framework is installed.
